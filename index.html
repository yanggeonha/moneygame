📋 설정 방법 (한 번만 하면 됩니다)
  1. Firebase 콘솔에 구글 계정으로 로그인
  2. 프로젝트 추가 → 이름 입력 (예: money-game) → Google Analytics는 끄기 → 만들기
  3. 왼쪽 메뉴에서 빌드 → Realtime Database 클릭 → 데이터베이스 만들기
  4. 위치는 그대로 두고 → 테스트 모드로 시작 선택 → 사용 설정
  5. 상단 규칙 탭 클릭 → 아래 규칙으로 변경 후 게시: 
    {
  "rules": {
    ".read": true,
    ".write": true
  }
}
    ※ 교실 게임용 임시 규칙입니다. 누구나 읽기/쓰기 가능. 더 안전하게 하려면 인증을 추가하세요.
  6. 좌측 상단 ⚙️ 프로젝트 설정 → 하단 내 앱에서 웹(</>) 아이콘 클릭
  7. 앱 닉네임 입력 → 앱 등록 → 표시되는 firebaseConfig 객체 복사
  8. 아래 칸에 붙여넣고 저장 클릭
